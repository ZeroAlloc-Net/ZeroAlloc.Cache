# ZC0002 — Reference-Type Cache Key Parameter

**Severity:** Warning

A method parameter that is used as part of the cache key has a reference type other than `string`. The generator calls `ToString()` on each key parameter at runtime to build the cache key. For reference types, `ToString()` defaults to the fully qualified type name (e.g. `MyApp.Filter`) rather than a value-based representation, which means all instances of that type map to the same cache entry regardless of their content.

---

## Example

```csharp
public sealed class ProductFilter
{
    public string? Category { get; init; }
    public decimal MaxPrice { get; init; }
    // ToString() not overridden — returns "MyApp.ProductFilter" for every instance
}

[Cache(TtlMs = 60_000)]
public interface IProductRepository
{
    // ZC0002: 'filter' is a reference type — ToString() may not be unique
    ValueTask<IReadOnlyList<Product>> SearchAsync(ProductFilter filter, CancellationToken ct);
}
```

All calls to `SearchAsync` with any `ProductFilter` instance will hit the same cache entry because the key is always `"MyApp.ProductFilter"`.

---

## How to fix

**Option A — Override `ToString()`** on the type to return a stable, unique, value-based string:

```csharp
public sealed class ProductFilter
{
    public string? Category { get; init; }
    public decimal MaxPrice { get; init; }

    public override string ToString()
        => $"{Category}:{MaxPrice}";
}
```

**Option B — Use a value type or record struct** so the generator can produce a stable key without a custom `ToString()`:

```csharp
public readonly record struct ProductFilter(string? Category, decimal MaxPrice);

[Cache(TtlMs = 60_000)]
public interface IProductRepository
{
    ValueTask<IReadOnlyList<Product>> SearchAsync(ProductFilter filter, CancellationToken ct);
}
```

`record struct` synthesizes a `ToString()` that includes all member values, which is both stable and unique.

**Option C — Flatten the key into primitive parameters** when the number of fields is small:

```csharp
[Cache(TtlMs = 60_000)]
public interface IProductRepository
{
    ValueTask<IReadOnlyList<Product>> SearchAsync(string? category, decimal maxPrice, CancellationToken ct);
}
```

---

## When to suppress

Suppress only if you have verified that `ToString()` on the type returns a value that is both **stable** (same output for equal instances across calls and process restarts) and **unique** (different output for non-equal instances). Suppress with a justification comment:

```csharp
#pragma warning disable ZC0002 // ToString() on ProductFilter is overridden and stable
ValueTask<IReadOnlyList<Product>> SearchAsync(ProductFilter filter, CancellationToken ct);
#pragma warning restore ZC0002
```
